🚀 Proceso de Desarrollo

1

Crear repositorio

Se crea un nuevo repositorio en GitHub, teniendo en cuenta las mejores prácticas para la nomenclatura y estructura inicial del proyecto.

2

Subir archivos

Se sube el código HTML y CSS al repositorio, organizando los archivos de manera lógica y manteniendo una estructura clara del proyecto.

3

Activar GitHub Pages

Se configura GitHub Pages desde la configuración del repositorio, seleccionando la rama principal como fuente para el despliegue automático.

4

Despliegue automático

GitHub automáticamente despliega la página web en GitHub.io, creando una URL pública accesible desde cualquier navegador web.
