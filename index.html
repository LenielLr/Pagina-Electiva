🎯 Crear página web usando GitHub.io

Objetivo: Crear un repositorio conteniendo una página web y utilizar GitHub.io para automatizar la creación de la página con el contenido del repositorio.

📑 Información del Proyecto

📂 Repositorio GitHub 🌐 GitHub Pages URL

🛠 Tecnologías Utilizadas

HTML5 CSS3 GitHub Pages

⚙️ Proceso de Desarrollo

  1. Crear repositorio: Nuevo repositorio en GitHub llamado “Electiva2”.
  2. Subir archivos: Crear y subir index.html y styles.css.
  3. Activar GitHub Pages: Settings → Pages → Deploy from branch.
  4. Despliegue automático: Página disponible en GitHub.io.
